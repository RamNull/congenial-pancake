# Changelog

All notable changes to the "Copilot Context Executor" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2025-11-14

### Added
- Initial release of Copilot Context Executor extension
- Command to execute Copilot prompts with full workspace context
- File picker for selecting code directory
- Input box for entering AI prompt payload
- Automatic workspace opening
- Integration with GitHub Copilot Chat
- Graceful fallback when Copilot Chat is not available
- TypeScript implementation with full type safety
- Test infrastructure with Mocha
- Comprehensive documentation

### Features
- Minimal context input (directory + prompt)
- Automatic workspace context passing to Copilot
- User-friendly error messages
- VS Code native UI integration
